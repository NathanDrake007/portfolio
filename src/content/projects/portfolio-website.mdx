# Portfolio Website

A modern, responsive portfolio website built with Next.js, TypeScript, and Tailwind CSS. This project showcases my work and provides a platform for sharing my thoughts through blog posts.

## Features

- Responsive design that works on all devices
- Dark mode support
- Blog system using MDX
- Project showcase
- Modern UI with smooth animations
- SEO optimized

## Technical Details

The website is built using Next.js 14 with App Router, providing excellent performance and SEO capabilities. TypeScript ensures type safety throughout the codebase, while Tailwind CSS enables rapid UI development with a utility-first approach.

The blog system uses MDX for content management, allowing for rich content creation with React components. Projects are also managed through MDX files, making it easy to update and maintain the portfolio.

## Development Process

The development process focused on creating a clean, maintainable codebase while ensuring excellent user experience. Key decisions included:

1. Using Next.js for its excellent performance and SEO capabilities
2. Implementing a dark mode system for better user experience
3. Creating a markdown-based content system for easy updates
4. Ensuring responsive design for all screen sizes
